# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-26

### Added
- Initial release
- Browser-to-phone calling with Twilio
- Browser-to-browser calling with WebRTC
- User authentication (email/password + Google OAuth)
- Credit system with daily bonuses
- Call history and analytics
- Admin dashboard
- User settings and profile management
- Real-time call status updates
- Call timer and duration tracking
- Mute/unmute functionality
- 150+ countries support
- Rate limiting and spam protection
- Dark mode UI with glassmorphism
- Responsive design for all devices
- Security features (JWT, password hashing, device fingerprinting)
- PostgreSQL database with Prisma ORM
- Redis caching layer
- Docker containerization
- Deployment guides for Vercel/Railway

### Security
- End-to-end encryption for calls (WebRTC)
- Secure password hashing with bcrypt
- JWT-based session management
- API rate limiting
- CORS protection
- Input validation and sanitization

### Performance
- Optimized images with Next.js
- CSS-in-JS with TailwindCSS
- Code splitting and lazy loading
- Redis caching
- Database query optimization
